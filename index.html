Mis trabajos en el BootCamp

Landing Page

En este proyecto, creamos una landing page utilizando CSS. El objetivo principal fue demostrar los estilos que se pueden aplicar al HTML, como cambios en el color de texto, fondos y otros elementos visuales. A través de este trabajo, se evidencian las diversas opciones de personalización que ofrece CSS para mejorar la apariencia y funcionalidad de una página web.

Iguana Page

En este proyecto, creamos una página web cuyo objetivo principal fue trabajar con Flexbox. El propósito fue entender y explorar todas las opciones que Flexbox ofrece, así como familiarizarnos con sus comandos, para utilizarlos de manera eficiente y adecuada en nuestros futuros proyectos.

Page

En este proyecto, utilizamos Bootstrap, una herramienta que permite maquetar páginas web de manera rápida y sencilla. Al ser de código abierto, se puede utilizar de forma gratuita y sin restricciones. Gracias a su amplia variedad de componentes y estilos predefinidos, pudimos explorar su potencial y demostrar la rapidez con la que es posible desarrollar una página web utilizando esta herramienta.
